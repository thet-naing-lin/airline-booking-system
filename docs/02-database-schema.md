# Database Schema

## Relationships
```text
User 1 --- many Booking
Booking 1 --- many Passenger
```

## users
Use Laravel default users table for MVP.

| Column | Type | Notes |
|---|---|---|
| id | bigint | primary key |
| name | string | staff name |
| email | string unique | login identifier |
| password | string | hashed |
| role | string | MVP values: `admin`, `staff` |
| timestamps | timestamps | Laravel default |

## bookings
| Column | Type | Notes |
|---|---|---|
| id | bigint | primary key |
| booking_code | string unique | internal reference, generated by system |
| pnr | string(20), nullable, indexed | airline/supplier reference |
| departure_location | string | required |
| destination | string | required |
| travel_date | date | required |
| travel_time | time, nullable | optional |
| airline_name | string, nullable | optional in MVP |
| flight_number | string, nullable | optional in MVP |
| contact_name | string | required |
| contact_phone | string | required |
| deposit_amount | decimal(12,2) | default 0 |
| total_amount | decimal(12,2) | default 0 |
| status | string | pending, confirmed, travelled, cancelled, refunded |
| comment | text, nullable | internal note |
| created_by | foreignId users | required |
| timestamps | timestamps | Laravel default |

## passengers
| Column | Type | Notes |
|---|---|---|
| id | bigint | primary key |
| booking_id | foreignId bookings | cascade on delete |
| full_name | string | required |
| nrc_number | string, nullable | domestic identity information |
| date_of_birth | date, nullable | optional initially; required according to business rule later |
| phone_number | string, nullable | individual passenger contact |
| passport_number | string, nullable | international travel |
| ticket_number | string, nullable, indexed | individual ticket identifier |
| seat_number | string, nullable | individual seat |
| timestamps | timestamps | Laravel default |

## Constraints
- Do not add a persisted `passenger_count` column.
- `passengers_count` is calculated using Eloquent `withCount('passengers')`.
- Delete booking -> delete its passengers using cascade.
- Require at least one passenger when a booking is created.
- `pnr` is allowed to be null for pending bookings.
